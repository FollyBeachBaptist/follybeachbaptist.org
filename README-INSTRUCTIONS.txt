# CarePortal Online Registration Package

This package is built for the Folly Beach Baptist Church Jekyll/Minimal Mistakes website.

## Files included

- `_pages/careportal.md` — the complete registration page
- `_pages/careportal-thank-you.md` — confirmation page
- `assets/images/careportal-header.png` — CarePortal header image
- `ANNOUNCEMENTS-BLOCK.md` — replacement text and Register button for the Announcements page

## Upload the files to GitHub

1. Open the church repository:
   `FollyBeachBaptist/follybeachbaptist.org`
2. Click **Add file**, then **Upload files**.
3. Upload the folders from this package while preserving their folder names:
   - `_pages`
   - `assets/images`
4. Click **Commit changes**.

## Add the button to Announcements

1. Open `_pages/announcements.md` in GitHub.
2. Click the pencil icon to edit.
3. Find the current CarePortal section near the top.
4. Replace that section with the contents of `ANNOUNCEMENTS-BLOCK.md`.
5. Click **Commit changes**.

The registration page will be:

`https://follybeachbaptist.org/careportal/`

## One-time FormSubmit activation

The first time someone submits the form, FormSubmit will send an activation email to:

`info@follybeachbaptist.org`

Open that email and click **Activate Form**. This is required only once. After activation, new registrations will be delivered to that email address.
